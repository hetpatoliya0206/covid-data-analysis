# COVID-19 Data Analysis (India)

## Project Overview
This project analyzes COVID-19 data across Indian states to identify trends, high-risk regions, and key insights using Python.

## Objectives
- Analyze state-wise COVID-19 cases
- Track growth trends over time
- Identify top affected states
- Calculate death rates


## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib


## Key Features
- Data cleaning and preprocessing
- State-wise case analysis
- Daily case growth calculation
- Death rate analysis
- Data visualization using charts


## Visualizations

### 1. Total Cases Over Time
Shows growth of COVID-19 cases in India.

### 2. Top 10 States by Cases
Highlights most affected states.

### 3. Top 5 States Trend Comparison
Compares growth patterns across major states.

## 🔍 Key Insights
- Maharashtra had the highest number of COVID-19 cases.
- Southern states like Tamil Nadu and Andhra Pradesh showed rapid growth.
- Death rates varied significantly across states.
- Cases increased sharply during peak periods.
